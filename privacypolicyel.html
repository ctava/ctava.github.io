Μπορούμε να αποκαλύψουμε προσωπικές πληροφορίες (πληροφορίες που χρησιμοποιούνται για την αναγνώρισή σας ως άτομο), πώς να συλλέγουν αριθμό προπονητής σας, το όνομα, το email και τον αριθμό τηλεφώνου. Οι τύποι των προσωπικών πληροφοριών που συλλέγουμε μπορεί να ποικίλλει ανάλογα με τη χρήση των λειτουργιών από τις πλατφόρμες. 


Πληροφορίες που συλλέγουμε αυτόματα


δεδομένα χρήσης.  Κάθε φορά που επισκέπτεστε ή να αλληλεπιδρούν με τις πλατφόρμες που έχουμε, καθώς και τρίτους διαφημιζόμενους και / ή φορείς παροχής υπηρεσιών, μια ποικιλία τεχνολογιών που χρησιμοποιούν αυτόματο ή παθητικά να συλλέγουν πληροφορίες σχετικά με τον τρόπο με τον οποίο έχει πρόσβαση οι πλατφόρμες και να χρησιμοποιηθεί ( " δεδομένα χρήσης "). Πληροφορίες χρήσης όπως τον τύπο του browser, τον τύπο της συσκευής, λειτουργικού συστήματος, απελευθέρωση αίτησης περιλαμβάνουν, υπηρέτησε την πλευρά, ο χρόνος που προηγείται προβολές σελίδων, και η χρήση των λειτουργιών ή εφαρμογών στις πλατφόρμες, όπως η προβολή και κοινή χρήση των προϊόντων, τα οποία αλληλεπιδρούν με τις επαφές και την πρόσβαση στο Web -Sites.Diese πληροφορίες μας βοηθήσετε να βελτιώσουμε τις πλατφόρμες μας φρέσκο ​​και ενδιαφέρον για τους επισκέπτες μας και μας επιτρέπει να προσαρμόσουμε το περιεχόμενο με τα ενδιαφέροντα του επισκέπτη.


Αναγνωριστικό συσκευής  έχουμε αυτόματα τη διεύθυνση IP σας ή άλλο μοναδικό αναγνωριστικό ( "Αναγνωριστικό συσκευής") για τη συσκευή (υπολογιστή, κινητό τηλέφωνο, tablet ή άλλη συσκευή) που χρησιμοποιείτε για να αποκτήσετε πρόσβαση στην πλατφόρμα. Ένα αναγνωριστικό της συσκευής είναι ένας αριθμός που έχει εκχωρηθεί στη συσκευή σας όταν βρίσκεστε σε ένα site ή server της και τους υπολογιστές μας αναγνώριση της συσκευής σας με αναγνωριστικό συσκευής του. Μπορούμε να χρησιμοποιήσουμε το αναγνωριστικό συσκευής, μεταξύ άλλων, να διαχειρίζονται τις πλατφόρμες σε προβλήματα με τους servers, τις τάσεις, την ιστοσελίδα του χρήστη κομματιού μας για να αναλύσουμε τις κινήσεις, σε σας και καλάθι αγορών σας εντοπίσει, και να συγκεντρώσει ευρεία δημογραφικές πληροφορίες για συνολική χρήση.


μπισκότα? Pixel ετικέτες  Οι τεχνολογίες που θα μπορούσαν να χρησιμοποιηθούν στις πλατφόρμες για τη συλλογή δεδομένων χρήσης, συμπεριλαμβανομένης της αναγνώρισης της συσκευής περιλαμβάνουν, αλλά δεν περιορίζονται σε αυτά: τα cookies (αρχεία σε μια συσκευή, όταν χρησιμοποιείται για να παραστεί ότι οι πλατφόρμες), Κινητό λογισμικού Analytics και ετικέτες εικονοστοιχείων (διαφανές γραφικών, μερικές φορές ορίζεται ως web beacon ή παρακολούθησης φάρο, σε μια ιστοσελίδα ή σε ένα e-mail αναφέροντας ότι μια σελίδα ή e-mail που έμοιαζε). Τα cookies μπορούν επίσης να χρησιμοποιηθούν για να συνδέσουν με ιστοσελίδες κοινωνικής δικτύωσης όπως το Facebook και το Twitter και αν θέλετε, η αλληλεπίδραση επιτρέπει μεταξύ των δραστηριοτήτων σας στις πλατφόρμες και τη δραστηριότητά σας στους ιστότοπους κοινωνικής δικτύωσης, όπως εμείς ή οι προμηθευτές μας, μπορεί να να είναι ή τοποθέτηση παρόμοια αρχεία cookie στη συσκευή για την ασφάλεια, την ευκολία της πλοήγησης και να εξατομικεύσει την εμπειρία σας όταν επισκέπτεστε ιστοσελίδες μας (καθώς μας επιτρέπει να επιλέξετε ποια, διαφημίσεις ή προσφορές πιθανότερο είναι ότι θα σας αρέσει, με βάση τα ενδιαφέροντα σας προτίμησης, τοποθεσία ή δημογραφικά στοιχεία). Μια ετικέτα pixel μπορεί να πει το πρόγραμμα περιήγησης για να πάρετε το περιεχόμενο από έναν άλλο διακομιστή.


Για να μάθετε πώς να μειώσει ή να διαγράψετε τα cookies που έχουν εγκατασταθεί στο φάκελο cookie του προγράμματος περιήγησής σας ικανό τον αριθμό των cookies που λαμβάνετε από εμάς, είστε στο μενού βοήθειας του προγράμματος περιήγησης σας ή άλλες οδηγίες για να σας σημείο σύνδεσης του προγράμματος περιήγησης. Εάν απενεργοποιήσετε ή τη λήψη των cookies, παρακαλούμε να σημειώσετε ότι ορισμένες λειτουργίες και υπηρεσίες στις ιστοσελίδες μας δεν μπορεί να λειτουργήσει σωστά, γιατί δεν αναγνωρίζουμε την ικανότητα και να συνδεθούν με multitouch λογαριασμό Προϊόντα (ες) σας να είναι. Επιπλέον, οι προσφορές μπορούμε να προσφέρουμε, όταν μας επισκεφτείτε, μην προσαρμοσμένες να είναι σχετικές με εσάς ή τα ενδιαφέροντά σας.
